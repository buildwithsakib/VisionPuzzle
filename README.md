# VisionPuzzle

> **An AI-powered hand gesture puzzle game that transforms your webcam into an interactive puzzle experience using real-time hand tracking.**

VisionPuzzle combines computer vision, gesture recognition, and browser technologies to create a unique puzzle-solving experience. Using only your hands and a webcam, you can capture a photo, solve it as a jigsaw puzzle, save your creations, and track your performance.

---

## ✨ Features

* 🖐️ **Real-Time Hand Tracking** powered by MediaPipe Hands.
* 📷 **Gesture-Based Photo Capture** using hand framing and pinch gestures.
* 🧩 **Interactive Puzzle Gameplay** with drag-and-drop hand controls.
* 🎯 **Multiple Difficulty Levels**

  * Easy (2×2)
  * Medium (3×3)
  * Hard (4×4)
  * Extreme (5×5)
* ⏱️ **Live Puzzle Timer** and Best Time tracking.
* 🎮 **Move Counter** to measure solving efficiency.
* 🏆 **Achievements & Statistics** stored locally.
* 🎨 **Multiple Themes** including Dark, Light, Neon, and Retro.
* 🔊 **Configurable Settings**

  * Sound
  * Music
  * Camera Mirror
  * Animations
  * FPS Counter
* 🎥 **Gameplay Recording** with downloadable video.
* 🖼️ **Photo Strip Gallery** with downloadable snapshots.
* 🎉 **Completion Effects** including animations and confetti.
* 💾 **Local Storage Support** for settings, records, and statistics.
* 📱 **Responsive Design** optimized for desktop and modern browsers.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/VisionPuzzle.git
```

Move into the project directory:

```bash
cd VisionPuzzle
```

Start a local server:

```bash
npx serve .
```

Then open the displayed localhost URL in your browser and allow camera access.

## 🎮 Controls

| Action            | Gesture / Key                                |
| ----------------- | -------------------------------------------- |
| Capture Photo     | Frame with both index fingers and hold pinch |
| Drag Puzzle Piece | Pinch and move your hand                     |
| Pause / Resume    | Space                                        |
| Reset Puzzle      | R                                            |
| Fullscreen        | F                                            |
| Mute              | M                                            |
| Close Dialog      | Esc                                          |
